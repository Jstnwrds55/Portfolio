About Me

I'm a software developer based in Denver, CO. I Primarily build Android apps with Kotlin, but I like to keep up with other stacks like Flutter/Dart, Django/Python, React/JS, and most recently Jetpack Compose. In my free time I like to write guitar, lift weights, and learn how stuff works.

Experience

I've been working as a software developer for a year and a half, but my passion for software goes back much further. In middle school I picked up Blender which I used to 3D model and animate, though I never felt like 3D modelling was my thing. In high school I picked up HTML/CSS & Javascript which led to me tinkering with iOS development, and I eventually I learned to really program with a book teaching Python. During college, I learned more of the fundamentals, graduated, and now work full time writing sofware.

Projects

Todo App

Fully responsive mutli-list todo application built using React and Material UI for the frontend and Firebase for user authentication, data storage, and hosting.

Mahjong Game

Fully featured Mahjong game written in Java using Swing for the GUI. Makes use of advanced object-oriented programming concepts as well as custom data structures.

Epidemic Simulator

Epidemic simulator written in Python using Glowscript with VPython, a Python library that makes simulating 3d objects straight forward. For this project, I wrote a lab manual while writing the program that documents the steps to reproduce.

Blackjack Game

Blackjack game written in vanilla HTML/CSS and Javascript. Cards are created using basic styles & unicode characters and the game state is kept track of in the js file.

iOS Calculator App

Fully featured calculator app built with XCode and written in Swift. Behaves like a normal calculator allowing chained operations and operation history.

Selection Sort/Binary Search Visualizer

This project uses plain HTML/CSS/JS to show a visualization of a selection sort and subsequently allows the user to binary search for a value, also visualized. I used promises to achieve this.
